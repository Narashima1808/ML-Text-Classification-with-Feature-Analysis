# ML-Text-Classification-with-Feature-Analysis


**Interpretable NLP using Classical Machine Learning**

## 📌 Overview

This project builds and analyzes text classification models using **classical machine learning techniques**.

Unlike typical implementations, this project focuses on:

* **Feature engineering impact**
* **Model interpretability**
* **Error analysis**

---

## 🎯 Objectives

* Compare different feature representations:

  * Bag-of-Words (BoW)
  * TF-IDF (Unigrams)
  * TF-IDF (Bigrams)
* Train multiple models:

  * Naive Bayes
  * Logistic Regression
  * Support Vector Machine (SVM)
* Analyze:

  * Feature importance
  * Model errors
  * Confusion matrices

---

## 🧠 Research Focus

* How do features influence model performance?
* Which words drive classification decisions?
* Where do models fail and why?

---

## ⚙️ Methodology

### 1. Preprocessing

* Lowercasing
* Stopword removal
* Tokenization

### 2. Feature Engineering

* BoW
* TF-IDF (unigrams & bigrams)

### 3. Model Training

* Naive Bayes
* Logistic Regression
* Linear SVM

### 4. Evaluation

* Accuracy, Precision, Recall, F1-score
* Confusion Matrix
* Cross-validation
